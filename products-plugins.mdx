# 🛒 Plugin Marketplace – Extend Your Store with Powerful Features

The **Plugin Marketplace** in WC Studio is your one-stop shop to discover and install the extensions you need to grow your store. From essential WooCommerce add-ons to advanced marketing, analytics, and logistics tools — it's all here, ready to install with a click.

---

## 📍 Go to: WC Studio → Products → Plugins

---

## 🧭 What's Inside the Plugin Marketplace?

### 🌟 Featured Plugins

Hand-selected by WC Studio for reliability, performance, and relevance. These include:

- Payment gateways (Stripe, PayPal, Apple Pay)
- Shipping integrations (ShipStation, DHL, FedEx)
- SEO tools (Yoast, Rank Math)
- Conversion boosters (cart upsells, urgency timers)
- Security & optimization plugins
- More

---

### 📦 Plugin Categories

Quickly browse plugins by category:

- 📈 Marketing & SEO
- 💳 Payments & Gateways
- 🚚 Shipping & Logistics
- 🛍️ Sales & Promotions
- 📊 Analytics & Reports
- 🧰 Utility & Developer Tools

Use tags and search to filter by use case, compatibility, or popularity.

---

## 🔍 Discovering & Installing Plugins

1. Use the search bar or browse categories.
2. Click on a plugin to view details:
   - Description
   - Key features
   - Developer info
   - Compatibility
3. Click **Install** to add the plugin to your store.
4. Activate from the plugin list.

> ✅ No manual uploads or FTP required — everything installs in a single click.

---

## 🧩 Third-Party Integrations

Some plugins are developed by **trusted third-party providers**. These are clearly marked and may offer:

- Premium licenses
- External dashboard connections
- SaaS service integrations

You can still install and manage them fully within WC Studio.

---

## 🛠️ Manage Installed Plugins

After installation:

- Go to **Plugins → Installed**
- Filter by **Active**, **Inactive**, or **Update Available**
- Manage plugin state or delete as needed

> 🧠 For major feature changes, take a Snapshot before activating.

---

## 💡 Pro Tips

- ✅ Start with essential plugins like security, backups, and SEO
- ✅ Don’t overload — only install what adds value to your store
- ✅ Check plugin update frequency and compatibility before installing
- ✅ Use Snapshots to test plugins safely

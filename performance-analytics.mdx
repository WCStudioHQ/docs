# 📊 Performance – Analytics

The **Analytics** section in WC Studio gives you powerful insights into your store’s performance — including load speed, storage usage, bandwidth consumption, and user experience metrics.

Use this data to fine-tune your store, monitor usage, and ensure everything runs efficiently.

---

## 📍 Go to: WC Studio → Stores → Select a Store → Performance → Analytics

---

## 📈 What You’ll See in the Analytics Dashboard

### ✅ Store Speed Overview
- Real-time and historical performance scores
- Metrics for:
  - Time to First Byte (TTFB)
  - First Contentful Paint (FCP)
  - Largest Contentful Paint (LCP)
  - Total Blocking Time (TBT)
  - Cumulative Layout Shift (CLS)

---

### ✅ Page-Level Performance Reports
- Compare speed between homepage, product pages, checkout
- Identify slow-loading areas
- Get optimization suggestions

---

### 📊 Usage Metrics

These metrics show your store’s current server resource usage:

#### 💾 Total Storage Usage
- Displays total storage consumed by your store’s:
  - Media files
  - Themes & plugins
  - Database

> Example: `Total Storage Usage: 0 MB`  
Storage updates in real time as you upload new assets or install tools.

---

#### 🌐 Total Bandwidth Usage
- Shows monthly bandwidth used by your store
  - Includes images, page loads, scripts, and assets served to users

> Example: `Total Bandwidth Usage: 0 MB`  
Reset every billing cycle or based on your plan limits.

💡 **Pro Tip:** Monitor these metrics regularly to avoid hitting plan limits.

---

### 📱 Device & Network Breakdown
- Performance across desktop, tablet, and mobile
- Network-based performance diagnostics (e.g. slow mobile 3G/4G)

---

### 📉 Historical Trends
- Track performance over time
- Compare performance before and after changes
- Analyze peak-hour behavior during promotions
